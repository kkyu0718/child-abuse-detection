# Main Feature

1. 아동학대 지속을 방지하고, 즉시 아이를 보호할 수 있도록 **실시간 아동학대 감지 및 신고프로젝트**
2. 폭력이 일어나면 관할 경찰서에 신고하고, 학대 정황이 모호하다고 판단하면 그 영상을 **전문가에 의해 학대 정황이 확인**
